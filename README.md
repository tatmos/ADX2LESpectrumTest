ADX2LE スペクトラムテスト  

このサンプルにはADX2LEのランタイムライブラリ(.lib .aなど)は含まれていません。

unity/plugin/criware_unity_plugin.unitypackage
を実行してランタイムライブラリをインストールしてください。
